# vue-covid19
  Vue组件开发---疫情展示
## 依赖第三方插件
   1. axios
   2. echarts
   3. swiper
## 组件
   1. header  头部图片
   2. info    展示病毒信息
   3. CaseNum 统计数据--axios请求数据
   4. 疫情地图
      echarts+api接口（天行数据(不跨越)+新浪数据（跨域））
   5. TreeChart  疫情趋势图
   6. News    新闻列表
   7. Footer  尾部图片
## 打包
   `npm run build`
   下载本地服务  `npm i -g serve`
   运行dist目录  `serve -s dist`
